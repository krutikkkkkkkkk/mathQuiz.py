# mathQuiz.py
Math Quiz game in Python3

run `mathGameV1.py`

Game starts with `10 points`

On every correct answer : `+5 points` and on every incorrect answer: `-5 points`

### Quiz: 50 Random Questions

- [x] Addition
- [x] Subtraction
- [x] Multiplication
